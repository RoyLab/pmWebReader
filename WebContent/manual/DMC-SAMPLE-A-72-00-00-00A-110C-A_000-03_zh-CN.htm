1  N1 SET 外圈旋钮（图1[1]）

    AUTO-
  • 两个 N1 基准游标由 FMC根据 N1 极限页面和起飞基准页面调定
  • N1 基准游标显示 A/T的有效 N1 极限。

    BOTH -
  • 两个 N1 基准游标和读数将通过人工旋转 N1 SET 内圈旋钮调定
  • 对 A/T 操作没有影响。

    1 或 2 -
  • 各自的 N1 基准游标和读数将通过人工旋转 N1 SET 内侧旋钮调定
  • 对 A/T 操作没有影响。

2  N1 SET 内圈旋钮（弹簧弹到中间位）（图1[2]）

旋转-当 N1 SET 外圈旋钮在 BOTH，1 或 2 时，调节 N1 基准游标和读数。

3  N1 基准游标（图1[3]）

显示（绿色）- 当 N1 SET 外圈旋钮在 AUTO，1，2 或 BOTH位置时显示。

4  N1 红线（图1[4]）

显示（红色）- N1% RPM 操作极限

5  N1 指令区（图1[5]）

显示（白色）- 实际 N1 和推力手柄位置指令值的瞬间差异。

6  N1 RPM 读数（数字）（图1[6]）

显示（红色）-

  • 超出操作极限
  • 在地面，发动机停车后，红色方框表示飞行中曾超出极限

显示（绿色） –

  • 当 N1 SET 外圈旋钮在 AUTO 时，显示空白
